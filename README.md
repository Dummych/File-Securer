# FS

FS (File Securer) — консольное приложение для безопасного хранения пользовательских данных.

## Использование

+ Создайте профиль с помощью /reg
+ Войдите в профиль через /login
+ Получите список команд через /help (мне лень писать здесь)
+ Data папка для хранения зашифрованных файлов
+ Enc папка для отправки файлов на шифрование
+ Dec папка для получения расшифрованных файлов

## Возможности

+ Регистрация
+ Авторизация
+ Криптография
+ ПОЛНОЕ Шифрование файлов и пароля
